Цікаве тестове завдання на знання SQL 

Есть таблица TAB:
    id  cnt
    1   5
    2   4
    3   7
    4   4

    Что будет в результате выполнения запроса
    SELECT max(id) over() FROM TAB 


    Ответ:

    4
    4
    4
    4

    Что будет в результате выполнения запроса
    SELECT count(NULLIF(cnt,5)) FROM TAB

    Ответ:

    3

    Что будет в результате выполнения запроса:
      SELECT count(id) FROM A
      GROUP BY cnt
      HAVING COUNT(*) > 1

    Ответ:

    2

В першому завданні SELECT max(id) over() FROM TAB використовується віконна функція. Віконна функція виконує обчислення для набору стрічок.
Документація про віконні функції


В другому завданні SELECT count(NULLIF(cnt,5)) FROM TAB використовується функція NULLIF(значение1, значение2) - повертає значення NULL, якщо значение1 дорівнює значение2; інакше вона повертає значение1.

Функція count обчислює кількість вхідних стрічок, для яких значення не рівно NULL. Значить запит поверне 3 бо в стрічці де cnt = 5 функція NULLIF(cnt,5) поверне NULL, відповідно функція count це значення не буде враховувати.

В третьому завданні SELECT count(id) FROM A GROUP BY cnt HAVING COUNT(*) > 1 використовується групування GROUP BY і фільтрування HAVING.
